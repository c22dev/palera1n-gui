# Palera1n-GUI
A simple Automator script to add a graphic interface to palera1n
## Features
### Included
- Auto-downloading palera1n from official repo
- Arguments support
- Ask for version
- Pyimg4 auto installer
### To come
- Auto-updater
- Normal mode support
- Logs (there is actually no log support in 0.1 beta)
- and more !
